## To Run Locally

#### Clone This Repo

```bash
git clone https://github.com/codernex/FileSure-Task-Management.git

cd FileSure-Task-Management

```

- Use pnpm package manager to install

```bash
pnpm install
```

- Create Environment Variable

```bash
 cp -r .env.example .env

 docker compose up
```

#### You are good to go
