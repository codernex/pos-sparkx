## Point Of Sale Solution

#### Using By Sparkx Fashion Limited

## To Run Locally

#### Clone This Repo

```bash
git clone https://github.com/codernex/pos-sparkx.git

cd pos-sparkx

```

- Use pnpm package manager to install

```bash
pnpm install
```

- Create Environment Variable

```bash
 cp -r .env.example .env

 docker compose up
```

#### You are good to go
